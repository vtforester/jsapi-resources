# Layout samples

Includes layout samples using ArcGIS Maps SDK for JavaScript and Calcite components

## Setup locally

To start each sample perform the following:

1. Navigate to one of the working directories (e.g., `cd interactive-filter-layout`)
   - [Layout](./interactive-filter-layout/)
   - [Dashboard](./dashboard-sample/)
   - [Finder](./finder-sample/)
2. Install dependencies: `npm i`
3. Start the development server: `npm run dev`
4. _Optionally_, if building and testing production apps, run `npm run build && npm run preview`
