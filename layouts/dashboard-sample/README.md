The dashboard sample demonstrates how to build a responsive, data-driven dashboard using the ArcGIS Maps SDK for JavaScript, Calcite Design System, and React.

This sample focuses on application layout and orchestration, showing how to manage application-level state, theming, and data flow in a dashboard-style app.

The app visualizes Global Power Plants data and combines maps, metrics, and charts into a cohesive user experience.

## Key features

- Using ArcGIS Maps SDK Web Components in a React application
- Building a dashboard layout with header, sidebar, metrics, map, and charts
- Querying a FeatureLayer and computing derived metrics
- Filtering data and updating metrics reactively
- Integrates Calcite components including Action Bar, Dialog, and theming
- Supports light and dark modes
- Creates a responsive layout using CSS Grid and Flexbox

## Running the app

1. Install dependencies: `npm i`
2. Start the development server: `npm run dev`
3. _Optionally_, if building and testing production apps, run `npm run build && npm run preview`
